What is Coding

Coding is a way to give a computer a specific set of instructions. They are very, very, very specific, so you need to tell them exactly what to do. They are written in HTML, which is like an alphabet, and computer languages (like C#, C++, Javascript, and Python.) Today, we are teaching HTML and Javascript, as they are the widest used languages on the internet.
